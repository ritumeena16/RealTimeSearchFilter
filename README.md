# 🔍 Real Time Search Filter

A responsive web application built using **HTML**, **CSS**, and **JavaScript**. This project dynamically displays profile cards with real-time search functionality.
Users can search profiles by **name** or **bio**, view profile details in a modal, and interact with Follow/Following buttons.

---

## 🚀 Features

- 🔍 Real-time search by name and bio
- 👤 Dynamic profile cards
- 📄 Profile modal popup
- ❤️ Follow / Following button toggle
- ⚡ Skeleton loading animation
- 📱 Responsive design
- 🎨 Blur overlay effect
- ✨ Smooth hover animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
